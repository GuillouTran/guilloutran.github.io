#### Introduction:

In this tutorial, you'll learn how to create a Raspberry Pi Chat Server with Rocket Chat. You will also learn how to create a reverse proxy so that you can use your domain and share your chat server with your friend\!

#### Requirement:

* A Raspberry Pi 4+ (You can also use 3+ or alternative)
* A SD Card with Rapsbian Buster
* A minimal knowledge of Linux
